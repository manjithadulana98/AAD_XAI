# Dataset & Tool Downloads (external)

This repo doesn't bundle datasets. Use official sources:

- SparrKULee (KU Leuven auditory EEG datasets landing page)
- Auditory Attention Detection Dataset KULeuven (Zenodo record)
- DTU EEG+audio dataset for auditory attention decoding (Zenodo record)
- mTRF Toolbox (GitHub / MATLAB Central)

Tip: keep datasets in a separate folder outside git, e.g. `data_raw/`.
